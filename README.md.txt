# Monthly QA Automation Project

## Project Goal
Automate monthly QA performance reports using Python and Pandas.

## Dataset
CSV containing:

agent_name
month
calls_handled
csat_score
defect_flag


## Features

Monthly average CSAT per agent
Total calls and defects per month
Visualizations of CSAT trends
Exported reports (CSV)


## Skills Demonstrated

Python & Pandas
Data cleaning and aggregation
Basic visualization
Automation-ready reporting
